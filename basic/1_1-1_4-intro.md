# 1.1 - 1.4: Intro Tasks

## Description

Four warmup tasks to get you oriented on the platform. Covers flag submission, setting a username, reading the FAQ, and joining the Discord server.

## Solution

Nothing to hack here, just onboarding. I worked through them top to bottom.

**1.1** - The scoreboard has a test submission field. Typed in the provided flag, hit submit, done.

**1.2** - Profile settings had a username field. Set it, got the flag back.

**1.3** - Read through the FAQ page. The flag was sitting in the text.

**1.4** - Joined the Discord server from the invite link on the platform. Flag was pinned in the welcome channel.

Took maybe five minutes total. Useful for understanding how flag submission works before things get harder.
